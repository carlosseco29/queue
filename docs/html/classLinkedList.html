Detailed Description

template<class T>
class LinkedList< T >

A templated class for a linked list.

This class tries to mimic the std::forward_list's interface as much as possible

Template Parameters
TThe type of values that the linked list will hold
Note
Only the methods needed for the operation of Queue.h have been implemented.
